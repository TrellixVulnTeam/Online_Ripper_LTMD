About

Created by Austin Pearce | My GitHub


This website was created for the sole educational value of trying out the Django framework. The features that can be seen here are made possible by the talented people behind the key Python modules this project uses such as youtube_dl, spotipy, and eyed3.

Song Ripper's main function is to rip the audio from YouTube videos from a provided link from the user to create downloadable mp3 files through the form of a easily-accessible website. The initial focus was to use a user-made, public YouTube playlist to download any newly added videos. By using youtube_dl's handy archive function, the server will check if new videos were added to an inputted playlist link to prevent the redownloading of audio files. Thanks for checking it out!
